## 51

## 52

`hp = hp2` 时，调用赋值运算符，会将 `hp2` 拷贝给 `rhs`，随后交换的 `rhs` 和 `hp` 的值，则 `hp` 将成为 `hp2` 的副本，其本身的值被销毁。  
`hp = std::move(hp2)` 是，调用赋值运算符，由于调用 `move` 函数，将其转化为右值，因此赋值给 `rhs` 时调用的是移动构造函数，因此 `rhs`
接管了 `hp2`，`hp` 将会和 `rhs` 交换，并接管 `hp2` 的值，而 `hp` 的值将被销毁。
